Welcome!

This tutorial is for those who are interested in doing some very basic data wrangling and analysis on a simple Qualtrics (or other web-based) survey dataset. Coding can look scary, but once you've got some basic tools in your kit, you'll gain a more intuitive grasp of how to use code-based statistical analysis software like R. If you've run your survey through Qualtrics, we start from data download and walk through different types of data organization and the analyses you may want to run.
